# Data

This folder stores Statcast data generated by the numbered notebooks.

- `raw/`: pitcher-level Statcast parquet caches from `pybaseball.statcast_pitcher`.
- `processed/`: engineered sequence-feature tables and batter-name lookups.

For the final deliverable, regenerate the full top-100 panel with:

```bash
python3 code/pitch_sequence_pipeline.py --top-n 100
python3 code/pitch_sequence_extension.py --top-n 100
```

If the public GitHub repo excludes large data files later, the notebooks can rebuild them from Statcast.
